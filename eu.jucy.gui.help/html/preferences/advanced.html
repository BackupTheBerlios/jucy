RegExps for files

!! important changing this may make jucy share nothing!!!

Includes: RegExp for files to be shared (useful for example to specify if you only want to share mp3s) Default is to share anything.

Excludes: Files you want o be excluded from sharing, like tmp files.

Max Hashspeed

Use this setting to limit the maximum hash speed. This is useful for example if your drives are connected over a network. (0 means no limit, any given value is the maximum speed in MiB/s)

Filelist Refresh interval

How often the filelist should be refreshed. (Use a value between 5 and 360 Minutes, default is 60)

Bind Address

Set an interface name to bind to an interface. Or use an IP to specify an interface. Jucy will use this interface for all outgoing connections.
